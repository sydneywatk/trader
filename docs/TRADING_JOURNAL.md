# Trading Journal

Log of paper (and eventually live) trades. Append new entries at the bottom. Keep notes short — pattern, unusual context, or deviation from system rules.

## Columns

- **Date** — entry date (YYYY-MM-DD)
- **Strategy** — SID / S&D / other
- **Ticker** — symbol
- **Direction** — long / short
- **Entry** — fill price
- **Stop** — initial stop price
- **Target** — initial target price (or exit rule if rule-based)
- **Exit** — fill price on exit
- **P&L** — dollar P&L after fees/slippage
- **Notes** — signal quality, blocker overrides, deviations, lessons
